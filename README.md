# Color Guessing Game

A simple guessing game where you try to match the computer's randomly chosen color.

## How to Play
1. The game shows 4 colors: red, green, blue, black
2. Enter your guess
3. The computer picks a random color
4. If they match, you win!
5. Keep playing or quit anytime

## How to Run
1. Make sure Python is installed
2. Run: `python colorguessinggame.py`
3. Have fun!
